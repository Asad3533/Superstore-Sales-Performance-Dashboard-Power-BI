# 📊 Superstore Sales Performance Dashboard – Power BI

An interactive Business Intelligence dashboard built using Power BI to analyze sales performance, profitability, customer trends, and regional business insights using the Superstore dataset.

---

# 🚀 Project Overview

This project transforms raw retail sales data into an interactive and visually engaging dashboard that helps businesses monitor performance and make data-driven decisions.

The dashboard provides insights into:
- Sales trends
- Profitability
- Regional performance
- Customer segments
- Product categories
- Order behavior

---

# ✨ Dashboard Features

## 📌 KPI Metrics
Track important business metrics including:
- 💰 Total Sales
- 📈 Total Profit
- 📦 Total Orders
- 📊 Profit Margin

---

## 📍 Sales Analysis
Analyze sales performance across:
- Regions
- Categories
- Customer Segments
- Time Periods

---

## 📅 Trend Analysis
- Monthly sales trends
- Profit growth tracking
- Seasonal performance analysis

---

## 🎛️ Interactive Dashboard
Users can dynamically explore the dashboard using:
- Region filters
- Category slicers
- Segment filters
- Time-based interactions

# 📈 Key Business Insights

✅ West region generated the highest overall sales  
✅ Technology category produced strong profitability  
✅ Sales peaked during Q4 periods  
✅ Some categories generated high revenue but low profit margins  
✅ Customer segments showed different purchasing behaviors
